Swift
# 機能概要

アプリケーションを使用する時、表で表示された任意のカテゴリを選択することで、そのカテゴリに対応した人気ランキングの上位4つのレシピを表示する。

レシピを選択すると、レシピで使用する食材、所要時間などの情報が表示され、指定された箇所をタップすることで、レシピを掲載しているWebページを表示する。
